# 자바스크립트로 투뎁스 메뉴 만들기

## 의식의 흐름

### 1. 배열로 메뉴 정의
```javascript
const mainMenu = [
    { title: '큰메뉴1', url: '/menu1', subMenu: [...] },
    { title: '큰메뉴2', url: '/menu2', subMenu: [...] }
]
```

### 2. HTML 문자열 생성 함수 만들기
- 배열을 `forEach`로 돌면서 HTML 문자열 생성
- 템플릿 리터럴로 HTML 태그 작성
- `innerHTML`로 한 번에 삽입

### 3. 호버 이벤트 처리
- `addEventListener('mouseenter')` → 서브메뉴 보이기
- `addEventListener('mouseleave')` → 서브메뉴 숨기기

### 4. DOM 로드 후 실행
```javascript
document.addEventListener('DOMContentLoaded', () => {
    renderMenu()      // 메뉴 생성
    initMenuEvents()  // 이벤트 등록
})
```

## 핵심 포인트
- 데이터(배열) → HTML 문자열 → DOM 삽입
- CSS에서 서브메뉴 기본 `display: none`
- JavaScript는 호버 시에만 `display: flex`로 변경
